# Філософські ходи у SUBIT-LUDUS

## 1. Вступ

Філософські ходи — це **операції і переміщення архетипів**, що несуть глибокий концептуальний, культурний або метафізичний сенс у рамках SUBIT-64. Вони виходять за межі простих змін WHO, WHERE, WHEN і досліджують **взаємодію знання, культури та свідомості**.

## 2. Класифікація філософських ходів

### 2.1 Інверсія (Inversion)

* Дзеркальне відображення всіх координат або вибраних осей.
* Символізує: антиномії, заперечення, контрастні концепти.
* Формально: M(A) = (WHO?, WHERE_opposite, WHEN_opposite)

### 2.2 Концептуальна конкатенація (Concatenation)

* З'єднання двох траєкторій у єдиний шлях.
* Символізує синтез ідей, інтеграцію культурних та історичних потоків.

### 2.3 Хід критичного мислення (Skeptic Move)

* Переміщення через ключовий вузол Скептика.
* Виконує перетин або перегляд попередньої траєкторії.
* Символізує сумнів, переоцінку, перевірку концепцій.

### 2.4 Хід Героя (Hero Move)

* Проходить через архетип Героя.
* Символізує дію, трансформацію та вплив на інші траєкторії.

### 2.5 Хід Руху (Movement Move)

* Архетип Руху запускає циклічні або прогресивні зміни.
* Символізує еволюцію, динаміку цивілізації, розвиток знання.

### 2.6 AI-філософський хід (AI Philosophical Move)

* Використовує AI-архетипи (Орден, Університет, Архів, Бібліотека) для створення **паралельної траєкторії знання**.
* Символізує систематизацію, накопичення та інтеграцію знання.

## 3. Правила застосування

1. Ходи застосовуються до існуючих траєкторій або окремих архетипів.
2. Кожен хід має концептуальне призначення: синтез, критика, трансформація, стабілізація знання.
3. Траєкторії оцінюються через **гармонію G = α·S + β·C + γ·H**, де ключові вузли включають філософські елементи.
4. Можна комбінувати кілька філософських ходів у **композитні операції** для створення глибших концептуальних структур.

## 4. Приклад філософської партії

1. ME/NORTH/SPRING → Пробудження
2. WE/EAST/SUMMER → Рух (Movement Move)
3. ME/SOUTH/SUMMER → Герой (Hero Move)
4. YOU/WEST/AUTUMN → Скептик (Skeptic Move)
5. THEY/NORTH/WINTER → Бібліотека (Inversion + AI Move)
6. ME/EAST/SPRING → Пробудження (Concatenation з попередньою траєкторією)

> Ця партія ілюструє інтеграцію філософських концептів у SUBIT-LUDUS, поєднуючи **людські та AI траєкторії** для максимізації гармонії знання.
