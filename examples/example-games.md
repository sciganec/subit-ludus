# Приклади ігор SUBIT-LUDUS

## 1. Демонстраційна партія (12 ходів)

**Мета:** показати взаємодію людства та AI через архетипи

| Хід | Архетип                         | Опис трансформації                            |
| --- | ------------------------------- | --------------------------------------------- |
| 1   | THEY/WEST/AUTUMN → Догма        | Початок, сучасна система                      |
| 2   | ME/NORTH/WINTER → Самітник      | Введення індивіда, Cᴡ: THEY→ME                |
| 3   | ME/EAST/SPRING → Пробудження    | Cʀ: NORTH→EAST, Cᵗ: WINTER→SPRING             |
| 4   | WE/EAST/SPRING → Рух            | Cᴡ: ME→WE, ключовий вузол                     |
| 5   | ME/SOUTH/SUMMER → Герой         | Cᴡ: WE→ME, Cʀ: EAST→SOUTH, Cᵗ: SPRING→SUMMER  |
| 6   | YOU/WEST/AUTUMN → Скептик       | Cʀ: SOUTH→WEST, Cᴡ: ME→YOU, Cᵗ: SUMMER→AUTUMN |
| 7   | THEY/NORTH/WINTER → Бібліотека  | M: дзеркало, ключовий вузол                   |
| 8   | THEY/NORTH/SPRING → Орден       | Cᵗ: WINTER→SPRING, AI початок                 |
| 9   | THEY/NORTH/SUMMER → Університет | Cᵗ: SPRING→SUMMER, AI розвиток                |
| 10  | THEY/NORTH/AUTUMN → Архів       | Cᵗ: SUMMER→AUTUMN, AI накопичення знання      |
| 11  | THEY/NORTH/WINTER → Бібліотека  | Cᵗ: AUTUMN→WINTER, AI стабілізація            |
| 12  | ME/EAST/SPRING → Пробудження    | M: дзеркало, цикл повертається до людини      |

**Підрахунок гармонії:**

* Симетрія S = 3
* Концептуальна глибина C = 4 (всі ключові вузли пройдені)
* Логічна послідовність H = 9
* Загальна гармонія G = S + C + H = 16

## 2. Мета ігор

* Побудова **гармонійних траєкторій знання**
* Інтеграція AI та людських архетипів
* Створення концептуальної «музики знання» через траєкторії

## 3. Інші приклади

1. **Цикл культури:** ME/NORTH/SPRING → WE/EAST/SUMMER → YOU/SOUTH/AUTUMN → THEY/WEST/WINTER
2. **Траєкторія AI-науки:** THEY/NORTH/SPRING → THEY/NORTH/SUMMER → THEY/NORTH/AUTUMN → THEY/NORTH/WINTER
3. **Людська ініціатива:** ME/EAST/SPRING → ME/SOUTH/SUMMER → ME/WEST/AUTUMN → ME/NORTH/WINTER

> Усі приклади демонструють принцип **логічного, симетричного та концептуально багатого переміщення через SUBIT-64**.
