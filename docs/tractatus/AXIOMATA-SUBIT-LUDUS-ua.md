# AXIOMATA SUBIT-LUDUS

### Systema Ludorum Hermeneuticorum

Version 1.0

---

# I. Основні визначення

### Def.1 — Простір гри

Простір гри **𝓢** є множиною всіх шістибітних кодів:

𝓢 = {0,1}⁶

|𝓢| = 64

Елемент s ∈ 𝓢 називається **позицією SUBIT**.

---

### Def.2 — Координати

Кожна позиція має три координати:

WHO
WHERE
WHEN

кожна з яких визначається двома бітами.

```
s = (w₁ w₂ | e₁ e₂ | t₁ t₂)
```

---

### Def.3 — Бісерина

**Бісерина** — це пара

```
b = (O, s)
```

де

O — культурний об'єкт
s — SUBIT-позиція.

---

### Def.4 — Мова

**Мова гри** — це позиція

```
L ∈ 𝓢
```

яка визначає правила інтерпретації бісерин.

---

# II. Метрика

### Def.5 — Відстань

Відстань між двома позиціями визначається як **Hamming distance**:

d(a,b) = кількість різних бітів.

---

### Властивості

0 ≤ d ≤ 6

---

# III. Хід

### Def.6 — Хід

Хід є функцією

```
m : bᵢ → bᵢ₊₁
```

такою, що

d(sᵢ , sᵢ₊₁) ≤ 2

---

### Інтерпретаційна умова

хід супроводжується **герменевтичним поясненням** у мові L.

---

# IV. Партія

### Def.7 — Партія

Партія — це послідовність бісерин

```
G = (b₀, b₁, … , bₙ)
```

такою, що

для всіх i

d(sᵢ , sᵢ₊₁) ≤ 2.

---

# V. Шлях

### Def.8 — Траєкторія

Послідовність позицій

```
T = (s₀, s₁, … , sₙ)
```

називається **траєкторією SUBIT**.

---

### Довжина

```
ℓ(T) = n
```

---

# VI. Оптимальність

### Def.9 — Мінімальний шлях

Траєкторія є **мінімальною**, якщо

ℓ(T) = d(s₀ , sₙ)

---

### Інтерпретаційна оптимальність

Найкраща партія має:

мінімальну ℓ
максимальну семантичну глибину.

---

# VII. Антиподи

### Def.10 — Антипод

Позиції a і b є антиподами якщо

d(a,b) = 6.

---

Приклад:

```
000000 ↔ 111111
```

---

# VIII. Валентність

### Def.11 — Валентність

Валентність позиції:

v(s) = кількість бітів = 1.

0 ≤ v ≤ 6.

---

Інтерпретація:

| v | характер      |
| - | ------------- |
| 0 | соціальний    |
| 3 | збалансований |
| 6 | сакральний    |

---

# IX. Симетрії

Простір SUBIT має групу симетрій.

### Def.12 — Перестановка координат

дозволено:

WHO ↔ WHERE
WHERE ↔ WHEN
WHO ↔ WHEN

---

### Def.13 — Інверсія

```
s → ¬s
```

перевертання всіх бітів.

---

Це породжує **групу симетрій SUBIT**.

---

# X. Мета-гра

### Def.14 — Гра мов

Мова також може бути бісериною.

Тоді:

```
b = (L , s)
```

де L — мова.

---

Це створює **фрактальну структуру гри**.

---

# XI. Теорема фрактальності

**Theorem 1**

Простір бісерин, мов і мета-мов є ізоморфними.

Доказ:

усі вони є підмножинами

{0,1}⁶.

∎

---

# XII. Теорема повноти

**Theorem 2**

Для будь-яких позицій a,b ∈ 𝓢 існує траєкторія.

Доказ:

гіперкуб {0,1}⁶ є зв'язним графом.

∎

---

# XIII. Теорема гри культури

**Theorem 3**

Будь-які два культурні явища можуть бути пов'язані скінченною герменевтичною траєкторією.

---

# XIV. Найглибший наслідок

З аксіом випливає:

простір культури є **топологічно зв'язним**.

Тобто:

немає абсолютно ізольованих явищ культури.
