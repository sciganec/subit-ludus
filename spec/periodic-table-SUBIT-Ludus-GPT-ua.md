# Геометрія таблиці

Ця таблиця має дуже красиві властивості.

### 1. Вертикалі — тип мислення

Наприклад:

**НАРАТИВ**

тотеми
↓
герої
↓
нації
↓
ідентичності

це **еволюція історичної свідомості**.

---

### 2. Горизонталі — джерело істини

Наприклад:

**РОЗУМ**

міф
↓
логос
↓
історія
↓
наратив

це **еволюція раціональності**.

---

### 3. Діагоналі — цивілізаційні архетипи

Наприклад:

```
мова сну
↓
мова теофанії
↓
мова переживання
↓
мова присутності
```

це **розвиток містичного досвіду**.

---

# Центральна клітина таблиці

Найбільш центральна позиція:

```
РОЗУМ
ЧИСЛО
МОДЕРН
```

**мова математики**

Це символічно дуже точно:

математика — **найуніверсальніша мова цивілізації**.

---

# Найбільша антипара

У таблиці видно крайні точки:

|                    |          |
| ------------------ | -------- |
| мова ідентичностей | мова сну |

Це протилежні полюси культури:

соціальний конструктивізм
vs
містичне переживання.

# Що це означає для гри

Тепер **гра в бісер** має чітку структуру.

1️⃣ Гравці обирають клітину таблиці
(мову)

2️⃣ Кодують явища SUBIT-кодами

3️⃣ Будують **шлях по таблиці мов**

4️⃣ Отримують **герменевтичну історію**
