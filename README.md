## Про проєкт

**subit-ludus** — це спроба формалізувати містичну «Гру в бісер» з однойменного роману Германа Гессе, використовуючи концепцію SUBIT-64. Ми пропонуємо розглядати SUBIT-64 не як чергову знакову систему, а як **топологію розрізнень** — універсальний простір, у якому будь-яке явище культури (симфонія, картина, наукова теорія) може бути представлене як точка з певними координатами.

Проєкт поєднує:
- **Філософію** — герменевтичний аналіз природи символічних систем
- **Математику** — теорію 6-вимірного гіперкуба та метрику Геммінга
- **Гру** — практичні правила для проведення партій у «Гру в бісер»

## Основна ідея: Що таке SUBIT?

### Наближення перше: біт і субіт

Якщо **біт** — це елементарний **акт розрізнення** (так/ні, світло/темрява, я/не-я), то **субіт** — це структура, де цей акт може відбутися. SUBIT-64 є простором усіх можливих розрізнень.

### Наближення друге: три координати буття

Будь-яке питання можливе лише за присутності трьох фундаментальних координат:

| Координата | Питання | Можливі значення |
|------------|---------|------------------|
| **WHO** | Хто? | Я (ME), Ми (WE), Ти (YOU), Вони (THEY) |
| **WHERE** | Де? | Схід (EAST), Південь (SOUTH), Захід (WEST), Північ (NORTH) |
| **WHEN** | Коли? | Весна (SPRING), Літо (SUMMER), Осінь (AUTUMN), Зима (WINTER) |

**SUBIT = WHO × WHERE × WHEN** = 4 × 4 × 4 = **64** = 2⁶

### Наближення третє: біт як відповідь

Субіт — це структурна можливість для біта виявити себе. Біт — це відповідь на питання, яке ставить субіт. Технічно ми маємо по чотири варіанти відповідей на кожне з трьох питань, загалом — 64 комбінації.

### Наближення четверте: субіт і хаос

Субіт любить хаос у тому сенсі, що він **харчується ним**; субіт структурує хаос, продукуючи з нього біти:

> **Хаос → субіт → біт**

Біт неможливий без субіту і хаосу: субіт — його структурна необхідність, хаос — феноменологічна.

### Наближення п'яте: ізоморфність

Субіт приречений на ізоморфність у силу своєї універсальності. 64 комбінації SUBIT ізоморфні:

- 6-бітним двійковим числам: {0,1}⁶
- 64 гексаграмам «І-Цзин»: 8 триграм × 8 триграм
- Системі «інь-ян» у шести вимірах: {Yin, Yang}⁶
- 64 кольорам: {Blue, Green, Lime, Yellow, Orange, Red, Burgundy, Purple}²
- 64 архетипам колективного несвідомого

### Наближення шосте: порядок бітів

Порядок бітів у кристалі субіту **принциповий**. Кожна координата кодується двома бітами:

| WHO | Код | WHERE | Код | WHEN | Код |
|-----|-----|-------|-----|------|-----|
| ME (Я) | **10** | EAST (Схід) | **10** | SPRING (Весна) | **10** |
| WE (Ми) | **11** | SOUTH (Південь) | **11** | SUMMER (Літо) | **11** |
| YOU (Ти) | **01** | WEST (Захід) | **01** | AUTUMN (Осінь) | **01** |
| THEY (Вони) | **00** | NORTH (Північ) | **00** | WINTER (Зима) | **00** |

Бісерина — це 6-бітне слово у форматі **[WHO₁ WHO₂ WHERE₁ WHERE₂ WHEN₁ WHEN₂]**.

## Гра як архетипна трансформація

**Відстань Геммінга** між двома бісеринами — кількість бітів, якими вони відрізняються — визначає мінімальну кількість кроків для переходу між явищами.

**Архетипна трансформація** — це зміна рівно одного біта, що відповідає переходу від однієї фундаментальної характеристики до іншої (наприклад, від ME до THEY, від EAST до WEST, від SPRING до AUTUMN).

**Мета гри** — знайти шлях від явища A до явища B, де кожна проміжна точка має осмислену інтерпретацію в обраній мові архетипів.

---

## Герменевтичний договір: вибір мови гри

На початку кожної партії гравці обирають одну з 64 **мов архетипів**. Кожна мова визначається трьома координатами:

- **Джерело мови (WHO мови)** — сакральне, природа, розум, колектив
- **Спосіб структурації (WHERE мови)** — символ, знак, число, наратив
- **Часова належність (WHEN мови)** — архаїка, класика, модерн, постмодерн

Приклади мов: юнгіанські архетипи, алхімічна символіка, мова геральдики, мова генетичного коду, мова сновидінь тощо.

## Приклад партії

**Мова:** Юнгіанські архетипи  
**Початок:** Бетховен, Симфонія №5 — `100111` (ME, WEST, SUMMER) → архетип **Героя**  
**Кінець:** Малевич, «Чорний квадрат» — `000000` (THEY, NORTH, WINTER) → архетип **Самості**

**Шлях:**
1. `100111` → `000111`: Герой → **Тінь** (колективне несвідоме)
2. `000111` → `000011`: Тінь → **Мудрий Старець** (позачасова мудрість)
3. `000011` → `000001`: Мудрий Старець → **Трікстер** (гра, іронія, перехід)
4. `000001` → `000000`: Трікстер → **Самость** (цілісність)

Інтерпретація: індивідуальна творчість через зустріч із колективним, через мудрість і гру приходить до цілісності, яку втілює «Чорний квадрат» як мандала XX століття.

## Встановлення та використання

```bash
# Клонувати репозиторій
git clone https://github.com/sciganec/subit-ludus.git
cd subit-ludus

# Встановити залежності (поки що тільки Python 3.8+)
pip install -r requirements.txt

# Запустити приклад
python src/examples/game_beethoven_malevich.py
```

### Залежності
- Python 3.8 або вище
- (Опціонально) `numpy` для роботи з векторами
