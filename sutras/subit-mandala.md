# Геометрія цієї мандали

Це не просто таблиця.

Це **гіперкуб практик**.

* горизонталь → напрям енергії
* вертикаль → фаза
* кола → рівень свідомості

Тобто:

**тантрична практика = рух по SUBIT-мандалі.**
